一、内联文本语义

使用 HTML 内联文本语义(Inline text semantics)定义语句，结构，可以是一个词，一段，或任意风格的文字。

元素：a
描述：HTML a 元素 (或锚元素) 可以创建一个到其他网页、文件、同一页面内的位置、电子邮件地址或任何其他URL的超链接。

元素：abbr
描述：HTML abbr元素代表缩写，并可选择提供一个完整的描述。

元素：b
描述：HTML b元素表表示相对于普通文本字体上的区别，但不表示任何特殊的强调或者关联，通常以粗体显示。

元素：bdi
描述：HTML bdi 元素 (双向隔离元素) 会隔离可能以不同方向进行格式化的外部文本。

元素：bdo
描述：bdo 元素 (HTML双向覆盖元素)用于覆盖当前文本的朝向，它使得字符按给定的方向排列。

元素：br
描述：HTML 元素 换行 br 在文本中产生一个换行（回车键）。这对于写诗或写一个地址来说显得很有用。它可以将行明显地分开。

元素：cite
描述：HTML引用（ Citation）标签 (cite) 表示一个作品的引用。它必须包含引用作品的符合简写格式的标题或者URL，它可能是一个
根据添加引用元数据的约定的简写形式。

元素：code
描述：HTML code 元素呈现一段计算机代码. 默认情况下, 它以浏览器的默认等宽字体显示.

元素：data
描述：HTML data 元素 将一个指定内容和机器可读的翻译联系在一起。但如果内容是与 time 或者 date 相关的，一定要使用 time。

元素：dfn
描述：HTML 定义元素 (dfn) 表示术语的一个定义。

元素：em
描述：HTML 着重元素 (em) 标记出需要用户着重阅读的内容，em 元素是可以嵌套的，嵌套层次越深，则其包含的内容被认定为越
需要着重阅读。

元素：i
描述：HTML元素 i 用于表现因某些原因需要区分普通文本的一系列文本。例如技术术语、外文短语或是小说中人物的思想活动等，它的内容通常以斜体显示

元素：kbd
描述：HTML键盘输入元素(kbd) 用于表示用户输入，它将产生一个行内元素，以浏览器的默认monospace字体显示。

元素：mark
描述：这个 HTML mark 标签代表突出显示的文字,例如可以为了标记特定上下文中的文本而使用这个标签. 举个例子，它可以用来显示搜索引擎搜索后关键词。

元素：nobr
描述： HTML nobr 元素阻止文本自动拆分成新行，所以它展示为长的一行，可能还需要滚动。这个标签不是标准的 HTML，并且不该使用。反之应该使用 CSS 属性

元素：q
描述：HTML引用标签 (q)表示一个封闭的并且是短的行内引用的文本. 这个标签是用来引用短的文本，所以请不要引入换行符; 对于长的文本的引用请使用 blockquote 替代.

元素：rp
描述：HTML rp 元素用于为那些不能使用 元素展示 ruby 注解的浏览器，提供随后的圆括号。

元素：rt
描述：HTML rt element 包含字符的发音，字符在 ruby 注解中出现，它用于描述东亚字符的发音。这个元素始终在ruby元素中使用。

元素：rtc
描述：HTML rtc 元素包含文字的语义注解，它们在 rb 元素中展示。rb 元素可以拥有发音 (rt) 和语义(rtc) 注解。

元素：ruby
描述：HTML ruby 元素 被用来展示东亚文字注音或字符注释。

元素：s
描述：HTML s 元素 使用删除线来渲染文本。使用 s 元素来表示不再相关，或者不再准确的事情。但是当表示文档编辑时，不提倡使用s ；为此，提倡使用 del 和 ins 元素。

元素：samp
描述：samp 元素用于标识计算机程序输出，通常使用浏览器缺省的 monotype 字体（例如 Lucida Console）。

元素：small
描述：HTML 中的元素將使文本的字体变小一号。(例如从大变成中等，从中等变成小，从小变成超小)。在HTML5中，除了它的样式含义，这个元素被重新定义为表示边注释和附属细则，包括版权和法律文本。

元素：span
描述：HTML span 元素是短语内容的通用行内容器，并没有任何特殊语义。可以使用它来编组元素以达到某种样式意图（通过使用类或者Id属性），或者这些元素有着共同的属性，比如lang。应该在没有其他合适的语义元素时才使用它。span 与 div 元素很相似，但 div 是一个 块元素 而 span 则是 行内元素 .

元素：strong
描述：Strong 元素 (strong)表示文本十分重要，一般用粗体显示。

元素：sub
描述：HTML sub 元素定义了一个文本区域，出于排版的原因，与主要的文本相比，应该展示得更低并且更小。

元素：sup
描述：HTML sup 元素定义了一个文本区域，出于排版的原因，与主要的文本相比，应该展示得更高并且更小。

元素：time
描述：HTML time 标签(time) 用来表示24小时制时间或者公历日期，若表示日期则也可包含时间和时区。

元素：u
描述：HTML u 元素使文本在其内容的基线下的一行呈现下划线。在HTML5中, 此元素表示具有未标注的文本跨度，显示渲染，非文本注释，例如将文本标记为中文文本中的专有名称(一个正确的中文标记), 或 将文本标记为拼写错误。

元素：var
描述： 标签表示变量的名称，或者由用户提供的值。

元素：wbr
描述：HTML wbr 元素 — 一个文本中的位置，其中浏览器可以选择来换行，虽然它的换行规则可能不会在这里换行。
