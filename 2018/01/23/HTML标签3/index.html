一、表单

HTML 提供了许多可一起使用的元素，这些元素能用来创建一个用户可以填写并提交到网站或应用程序的表单。详情请参阅 HTML forms guide。

元素：button
描述：HTML button元素 表示一个可点击的按钮。能够在表单，或者 HTML 文档的任一需要按钮的位置。默认情况下 HTML 按钮会呈现与用户主机相似的样式，基于 user agent。

元素：datalist
描述：HTML Datalist 元素 (datalist) 包含了一组option元素,这些元素表示其它表单控件可选值.

元素：fieldset
描述：fieldset 标签可以将表单内的相关元素分组。fieldset 标签会在相关表单元素周围绘制边框。

元素：form
描述：HTML form 元素 表示了文档中的一个区域，这个区域包含有交互控制元件，用来向web服务器提交信息。

元素：input
描述：HTML input 元素用于为基于Web的表单创建交互式控件，以便接受来自用户的数据。

元素：label
描述：HTML 元素表示用户界面中项目的标题。

元素：legend
描述：HTML的legend元素（也称为HTML的域说明元素（or HMTL Legend Field Element）代表一个用于表示它的父元素fieldset的内容的标题。

元素：meter
描述：HTML meter元素用来显示已知范围的标量值或者分数值。

元素：optgroup
描述：在一个web表单中, HTML元素 optgroup 会创建包含在一个 select 元素中的一组选项

元素：option
描述：在web表单中, HTML元素 option用于定义在select, optgroup 或datalist 元素中包含的项。option 可以在弹出窗口和 html 文档中的其他项目列表中表示菜单项。

元素：output
描述：HTML output标签表示计算或用户操作的结果。

元素：progress
描述：HTML中的progress (progress) 元素用来显示一项任务的完成进度.虽然规范中没有规定该元素具体如何显示,浏览器开发商可以自己决定,但通常情况下,该元素都显示为一个进度条形式.

元素：select
描述：HTML select (select) 元素是一种表单控件，可创建选项菜单。菜单内的选项为option , 可以由optgroup元素分组。选项可以被用户预先选择。

元素：textarea
描述：HTML textarea 元素表示一个多行纯文本编辑控件。

二、交互元素

HTML 提供了一系列有助于创建交互式用户界面对象的元素。

元素：details
描述：HTML details 元素被用作发现小部件，用户可以从其中检索附加信息。

元素：dialog
描述：HTML dialog 元素表示一个对话框或其他交互式组件，例如一个检查员或窗口。

元素：menu
描述：HTML menu 元素 呈现了一组用户可执行或激活的命令。这既包含了可能出现在屏幕顶端的列表菜单，也包含了那些隐藏在按钮之下、当点击按钮后显示出来的文本菜单。

元素：summary
描述：HTML summary 元素 用作 一个details元素的一个内容的摘要，标题或图例。

三、Web 组件

Web 组件是种近似 HTML（HTML-related） 的技术，这使得它能够，从本质上讲，创建和使用自定义元素，就好像它是普通的 HTML。此外，你甚至可以创建自定义版本的标准 HTML 元素。

元素：content
描述：HTML content元素— Web 组件 的技术套件的废弃部分 — 用于 Shadow DOM 内部作为 insertion point，并且不可用于任何正常的 HTML，现在已被slot元素代替，它在DOM 中创建一个位置，Shadow DOM 会插入这里。

元素：element
描述：element元素被定义在最新的 HTML DOM 元素中。

元素：slot
描述：HTML slot标签是web组件技术的一部分，slot是web组件的一个占位符，可以用来插入自定义的标记文本。可以创建不同的DOM树并进行渲染。

元素：template
描述：HTML template元素 是一种用于保存客户端内容的机制，该内容在页面加载时不被渲染，但可以在运行时使用JavaScript进行实例化。
